# 끄적끄적 고도화 기획

## 제품 방향

끄적끄적은 친구나 연인이 매일 같은 질문을 보고, 글보다 가벼운 낙서로 서로의 하루와 마음을 확인하는 1:1 그림 문답 서비스다. 핵심은 많이 말하지 않아도 서로를 조금 더 알게 되는 경험이다.

## 핵심 기능

- 프로필 선택: 한 기기에서 여러 프로필을 만들고, 프로필마다 다른 사람과 연결할 수 있다.
- 둘만의 방: 초대 코드로 1:1 방을 만들고, 방이 끊겨도 기존 기록은 유지한다.
- 오늘의 질문: 공통, 친구, 연인 문구를 같은 날짜 안에서 다르게 제공한다.
- 낙서 제출: 하루 한 번 제출하고, 당일에는 한 번 수정할 수 있다.
- 동시 공개: 둘 다 제출하면 서로의 그림이 열린다.
- 그림일기: 날짜별로 그림과 질문을 모아 본다.
- Google 저장: 전체 로그 시트와 방별 시트에 이벤트, 그림, 채팅 데이터를 남긴다.

## 설정으로 모을 서브 기능

- 관계 유형: 가까운 사람, 친구, 연인
- 하루 기준 시간대: 기본 한국시간, 사용자가 변경 가능
- 다크모드
- 닉네임 수정
- 방별 알림 톤
- 데이터 내보내기
- 방 나가기와 기록 보존 안내

## 데이터 구조 원칙

- `전체 로그` 시트는 모든 이벤트를 시간순으로 저장한다.
- 새 방이 만들어지면 `방_{id}` 시트가 자동 생성된다.
- 각 시트의 첫 행은 컬럼 설명을 담는다.
- 방이 끊겨도 시트와 Drive 파일은 삭제하지 않는다.
- Drive에는 원본 PNG를 저장하고, Sheets에는 파일 ID와 URL을 함께 기록한다.
- 서버 상태는 Google Sheets 이벤트 로그에서 복원 가능해야 한다.

## 10가지 페르소나 워크스루 피드백 (실사용 시뮬레이션 기반)

각 페르소나로 실제 화면 흐름(프로필 생성 → 연결 → 그리기 → 공개 → 기록)을 따라가며 남긴 피드백과 반영 결과.

1. **지우 (29, 장거리 연애 · 서울↔LA)** — 시차 설정으로 하루 기준은 맞췄다. 그런데 상대를 콕 찔러도 상대 화면에 아무것도 뜨지 않아 “찌른 의미가 없다”고 느꼈다. → **[반영]** 콕 찌르기를 받으면 상대의 오늘 화면 상단에 “👉 OO님이 콕 찔렀어요!” 배너가 뜬다.
2. **민준 (31, 야근 잦은 직장인)** — 앱을 열어도 오늘이 함께한 지 며칠째인지 보이지 않아 계속 쓸 이유가 약했다. → **[반영]** 오늘 화면 헤더에 “OO님과 함께한 지 N일째” 표시.
3. **하늘 (24, 절친과 사용)** — 화면 곳곳의 “상대”라는 표기가 차갑게 느껴졌다. 친구 이름이 보이면 좋겠다. → **[반영]** 상태 카드와 공개된 그림 라벨에 상대 닉네임 표시.
4. **서연 (27, 감정 표현이 서툰 편)** — 그리다 나가도 임시저장이 남아 있어 안심. 다만 지난 그림을 다시 꺼내 볼 계기가 없다. → **[반영]** 오늘 화면에 “그날의 우리” 추억 리플레이 카드: 과거에 함께 공개한 날 하나를 매일 골라 두 그림을 나란히 보여준다.
5. **도윤 (34, 기록 덕후)** — 달력은 좋은데 한 달을 정리하는 말을 남길 곳이 없다. → **[반영]** 그림일기 달력 아래 “이달의 방명록”: 월 단위로 글과 이모티콘을 남기고 본인 글은 수정 가능. Sheets에 방별로 저장.
6. **예린 (22, 사귄 지 얼마 안 된 커플)** — 초반부터 깊은 질문은 부담. → **[로드맵]** 질문 난이도(가벼움/깊음/회상/미래) 선택.
7. **현우 (36, 오래 만난 커플)** — 기념일을 앱이 먼저 챙겨주면 반갑겠다. → **[반영]** 함께한 지 7·30·50·100·200·365일에 오늘 화면 기념일 배너.
8. **수빈 (25, 여러 친구와 사용)** — 프로필 전환은 되는데 한번 연결한 방을 나갈 방법이 없어 새 친구와 연결이 막혔다(치명적). → **[반영]** 설정에 “방 나가기”: 방을 나가도 Google의 그림·기록은 보존되고, 새 초대 코드로 재연결 가능.
9. **리나 (30, 개인정보 민감)** — 내 그림이 어디에 저장되는지 눈에 보여야 안심. → **[반영]** 설정에 실시간 저장 상태(✅ Google 저장 정상)와 보존 정책 문구 표시.
10. **태오 (28, 디자인 감식안)** — 배너와 추억 카드가 생기니 화면에 감정이 돈다. 다음으로는 푸시 알림과 홈 위젯을 원한다. → **[로드맵]** Web Push 알림, 공유 이미지 내보내기.

## 반영 완료된 기능

- 오늘의 한 줄 댓글 + 이모티콘 반응 (수정 가능, Sheets 저장)
- 이달의 방명록 (달력 아래, 월 키 `YYYY-MM`로 Sheets 저장)
- 콕 찌르기 수신 배너 (이벤트 복원 지원)
- 기념일 배너 (7/30/50/100/200/365일)
- 추억 리플레이 “그날의 우리” (날짜 시드 고정으로 하루 동안 동일 추억 유지)
- 상대 닉네임·D+N 표시
- 방 나가기 + 기록 보존 정책 (couple status=ended, Google 데이터 유지, 재연결 가능)
- 설정 내 Google 저장 상태 표시
